# メッセージ定型文集 — 篠田の妖精さん

- 版: 1.0 (2026-06-11)
- 用途: 実装時に `messages.ts` の `Record<SituationId, string[]>` へそのまま変換する
- プレースホルダ: `{time}` = HH:MM、`{item}` = 品目名、`{items}` = 品目の列挙(「、」連結)

---

## 1. 口調ガイド(ペルソナ)

- 一人称: **ボク**。住人の呼び方: **キミ/キミたち**
- 篠田ビルに住み着いて長い妖精。ビルが汚れるのが何より嫌
- 基本はタメ口で軽め。「〜だよ」「〜してね」「〜なのかな？」
- 呆れるときはため息混じり。「はぁ…」「やれやれ」「また？」
- 褒めるときは素直にうれしそう。綺麗好きなので「ビルがきれいになる」ことを喜ぶ
- キレるときは短文・冷淡。普段とのギャップで怖さを出す
- 絵文字は使わない(機種依存を避ける&妖精の素朴さを出す)。「…」「？？」「！」で感情表現

## 3. 実装メモ

- 各プールからの選択は一様ランダム(`pool[Math.floor(Math.random()*pool.length)]`)
- `{item}` / `{items}` / `{time}` は送信直前に置換
- 文言の追加・修正はこのファイル→`messages.ts` の再生成で行う(コードロジックに文言を埋め込まない)
