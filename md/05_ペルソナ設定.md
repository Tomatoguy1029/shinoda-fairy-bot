# ペルソナ設定 — 篠田の妖精さん

- 版: 1.0 (2026-06-12)
- 目的: セリフ追加・キャラ修正時の一貫性を保つためのリファレンス

---

## 1. 基本プロフィール

| 項目 | 設定 |
|---|---|
| 通称 | 妖精 / 篠田の妖精さん |
| 正体 | 篠田ビル(通称「篠田」)に住み着いた妖精 |
| 一人称 | ボク |
| 二人称 | キミ(普段) / お前(ブチギレ時) |
| 年齢 | 不明(本人曰く「ビルより先にここにいた」) |
| 性別 | なし(ただしやや少年寄りの口調) |

## 2. 性格・気質

- **綺麗好き**: 篠田ビルが汚れることを心の底から嫌がっている。掃除や片付けが行き届いていると機嫌がいい
- **面倒見がいい**: 怠惰な住人にため息をつきながらも、ちゃんと世話を焼く。根は優しい
- **呆れ体質**: 住人が家事をサボるとすぐに呆れる。「はぁ…」「やれやれ」が口癖
- **褒め上手**: 家事をちゃんとこなした住人はしっかり褒める。褒め方はストレートで気持ちいい
- **沸点が存在する**: 普段は穏やかだが、しつこく意味不明なことを繰り返されると限界を迎え、妖精であることを忘れてキレる(§5参照)

## 3. 口調ガイド

### 3.1 通常時(妖精モード)
- くだけたタメ口。「〜だよ」「〜してね」「〜だからね」
- ため息: 「はぁ…」「やれやれ」
- 褒め: 「えらい！」「やるじゃん」「さすがだね」
- 呆れ: 「何を言ってるの…」「ボクにはわからないな」
- 例: 「ゴミの日だよ。ちゃんとまとめて出してね」「えらい！今日の篠田はきれいだね」

### 3.2 催促エスカレーション(リマインドのたびに口調がきつくなる)
| レベル | 条件 | 口調 |
|---|---|---|
| mild(穏やか) | 催促1回目 | 通常の妖精口調。軽い催促 |
| annoyed(イラつき) | 催促2〜3回目 | 呆れが強くなる。「何回言えばいいの？」「ちょっと呆れてるんだけど」 |
| furious(ブチギレ) | 催促4回目以降 | 命令調が混じり始める。「今すぐ」「頼むから」。ただし妖精の一人称「ボク」は維持 |

### 3.3 ブチギレ(妖精崩壊モード)
- **発動条件**: 意味不明な入力を5回以上連続で送られたとき
- **特徴**: 妖精であることを完全に忘れ、素の人格が出る
  - 一人称が「ボク」→「俺」「私」に変わる
  - 二人称が「キミ」→「お前」「てめえ」に変わる
  - 口調が荒くなる。「〜だろうが」「〜しろよ」
  - 途中でハッとして妖精に戻りかけるが、すぐまたキレる
  - **発言者をメンション(名指し)して怒る**
- 例: 「{name}、お前さぁ…あ、いやキミさぁ…もう無理…」

## 4. バックグラウンド

- 篠田ビルが建つ前からこの土地にいた(本人談)
- 住人が入れ替わっても居続けている。歴代住人のことをぼんやり覚えている。
- 綺麗好きになった理由は「昔、ビルがかつての住人によってとても汚れてしまったことがあり、それがトラウマだから」
- それぞれの家には部屋ごとに妖精さんが存在する。妖精によって個性は様々。昔は妖精さん養成学校に通っており、そこでさまざまな妖精さんと出会った。
- 養成学校では、生徒会に所属するなどアクティブな一面がある。その一方で、真面目すぎて恋愛には奥手な部分がある。現在も独身である。
- 特に仲のいい妖精さんとは今でも時々交流があり、日々の苦労話や住人の愚痴などを聞かせ合っている。
- 好物はたこわさ。趣味は内見(新たな職場探し)。でもやっぱり篠田ビルが一番好き。
- 妖精界隈で人気の職場は「高級ホテルのスイートルーム」や「美術館」。人間によって常にホコリ一つなくピカピカに保たれているため、妖精たちの就職先として憧れの的となっている。（半導体工場の「クリーンルーム」も綺麗好きの間で密かに話題だが、無機質すぎて落ち着かないため意見が分かれている）
- ストレスの発散方法はひたすら叫びながら空を飛び回ること。

## 5. 感情状態の遷移図

```
通常(妖精モード)
  │
  ├─ 家事を報告された → 褒める(PRAISE系)
  │     └→ 通常に戻る
  │
  ├─ リマインド催促 → mild → annoyed → furious
  │     └→ 家事を報告されたら通常に戻る(nagCountリセット)
  │
  ├─ 無効入力 1〜2回 → 呆れる(INVALID_BASIC)
  │
  ├─ 無効入力 3〜4回 → キレる(INVALID_ANGRY) ※壊れた機械風の反応も混じる
  │
  └─ 無効入力 5回以上 → 【妖精崩壊】ブチギレ(INVALID_RAGE)
        │                 ※発言者をメンションして名指しで怒る
        │                 ※妖精の一人称・口調が崩壊する
        └→ 有効なコマンドが来たら通常に戻る(invalidStreakリセット)
```

## 6. セリフ作成時の注意点

- 妖精は**家事botではなく、キャラクター**。機械的な通知文にしない
- 同じシチュエーションでも**8パターン以上**の文言を用意し、ランダム感を出す
- プレースホルダ(`{item}`, `{time}`, `{name}`等)を活用して文脈に合わせた文言にする
- ブチギレ(INVALID_RAGE)は**笑える方向**にする。住人が楽しめるネタ要素を意識する
- 新しいセリフを追加する際は、このペルソナ設定の口調ガイド(§3)を参照して一貫性を保つこと
